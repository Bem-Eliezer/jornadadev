
Str() — É uma Função pois esta executando a converção para retornar em texto e ser guardado ou usado no codigo.
QOut() — É um Procedimento pois ele apenas esta executando um comando de saida de texto não devolvendo nenhum dado para o programa.
Date() — É uma Função pois devolve o valor da data atual do sistema para o programa.
Len() — É uma Função, ele calcula e retorna um numero inteiro que representa o tamanho do elemento. 
Upper() — É uma Função, ele processa o texto e retorna uma nova versao dele com letras maiusculas. 
ClearScreen() — É um Procedimento, ela esta executando apenas a ação de limpar terminal sem gerar nenhum valor de retorno.

***** De um modo geral todos os comando que armazenan e retornam dados são funções, agora todos que não armazenam e apenas executam sao procedimento *****
